# Folio — Personal Knowledge Base

A beautiful, GitHub-backed personal notebook app with:
- 📓 Multiple notebooks & pages
- ✏ Rich text editing with inline formatting
- 💡 Annotations & comments
- ⌨ Syntax-highlighted code blocks
- 🕸 Knowledge graph view
- ✦ AI assistant (Gemini 2.0 Flash)
- 🌙 Dark / Light theme

## How to run

Open `index.html` directly in a browser, or serve locally:

```bash
python -m http.server 8080
```

Then visit **http://localhost:8080**

## Setup

You'll need a [GitHub Personal Access Token](https://github.com/settings/tokens) with `repo` scope.  
Enter it on first launch — your notes are saved to a private GitHub repo.
